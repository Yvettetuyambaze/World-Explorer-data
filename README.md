# World Explorer Data

This repository contains the source files and data assets for the World Explorer Data project. The purpose of this project is to provide a dataset combined with a web interface for exploring information about countries and places around the world.

This repository includes HTML, CSS, JavaScript, images, and supporting files used to build a web application that displays world data in an interactive format.

## Project Overview

The World Explorer Data project aims to:

- Provide visual exploration of world data
- Present country and location data in a structured way
- Create an interactive web interface for users to browse information

The repository does not include a detailed description yet, but it contains the project files necessary for deployment and further development.

## Setup Instructions

To use this project:

1. Clone the repository:

git clone https://github.com/Yvettetuyambaze/World-Explorer-data.git


2. Open `index.html` in a web browser.

3. Explore the web pages and interact with the content.

If you want to serve the site using a local web server (for example, to support PHP):

1. Install a web server like Apache, Nginx, or use tools like XAMPP or WAMP.
2. Place the repository folder in your server’s public directory.
3. Navigate to `http://localhost/World-Explorer-data/` to view the site.
